Build Plan

这个项目真正要开发什么

先把结构和同步做对，再做 fancy retrieval。

Phase 1

Capture / Store / Projections 结构、文件操作、模板、例子。

Phase 2

导入器、source refs、normalization、correction lifecycle。

Phase 3

Consolidate jobs、conflict handling、archive routing。

Phase 4+

platform sync、MCP、optional indexes、evolve pipeline。
